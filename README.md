üõí Shopping Cart

The Shopping Cart is where users can collect products from different stores for easy comparison and future reference. It also highlights the cheapest option to help users make smart purchasing decisions.

***Key Features***:

Add from Any Store: Users can add products from multiple stores directly into the cart.

Cheapest Option Highlighted: The system automatically highlights the store offering the lowest price for each product.

Easy Comparison: Compare prices of the same product across different sellers at a glance.

Save for Later: Keep track of desired products without losing them while browsing.

***How It Works***:

Add a product to the cart from any store while browsing or searching.

The cart automatically lists all added products and their prices from different sellers.

The cheapest option for each product is highlighted for easy identification.

Use the cart to compare prices, track deals, or plan purchases.
